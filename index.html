軟體工程師體驗營心得

兩個月前在facebook上看到六角學院的2024軟體工程師體驗營，仔細看居然是教從零打造個人品牌網站！

喚醒了幾年前想要寫網頁的一個小目標，透過自己的能力可以幫助到別人，無論是行銷還是個人品牌網站都好。

令我訝異的是居然只要$799 XD，想說這該不會是騙人的吧？怎麼那麼便宜？

看一下大綱居然有60小時的課程、助教會批改你寫的code、線上群組問答、校長直播授課還有作業做完又有證書？！

雖然我也不確定未來會不會成為軟體工程師，但誰說一定要是軟體工程師才會寫程式呢？

手刀報名下去之後覺得不對勁，30天要看60小時的內容+做作業+分組討論…完蛋了完蛋了…

校長說：只要你不放棄，六角學院就不會放棄你。

硬著頭皮上了啦！

第一週開始學了 VS Code + CodePen (好神奇的東西，居然可以在網頁上開發與測試網頁)

第二週的 Flexbox，這裡卡蠻久的，怎麼排都排不出來我要的樣子。網路上查資料、小組討論、看別人的code，才慢慢理解 XD

第三週的RWD，才發現原來是這樣做的，一個網頁要能適應電腦、平板、手機的螢幕大小，還以為是要各做一個網站，原來是用這招啊！

第四週開始用Hexo做自己的部落格，學了一點點指令跟git，從這裡開始進度完全落後…

所以樣板客製化的章節根本還沒來得及看 XD

除了寫程式之外我還學到了什麼？

  1. 校長從八點開始直播教學這周的進度，又一一解答同學們的問題直到深夜！這種負責任的態度，哪裡還找的到？(金句：只要你沒有放棄，六腳學院就不會放棄你)

  2. 群組討論區，只要有問題發問，校長、助教甚至是有經驗的同學都超踴躍的在解決問題！(金句：要勇於發問，因為你的問題就是同學的問題)

  3. 分組的幫助也很大，真的有依照程度去區分，彼此互相協助與解題。(金句：不用跟別人比較)
